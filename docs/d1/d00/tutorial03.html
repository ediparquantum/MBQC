Tutorial 3: 1D Odd size angles 0

Introduction

This tutorial demonstrates the creation of a 1D lattice in MBQC where each qubit is initialized with an H gate to be |+>, and each qubit has an angle of 0 on the z-axis. The goal is to show that with an odd number of qubits and specific flow and basis measurements, the final qubit should always be 0.

The source code can be foubd at tests/testOddNumberQubitAnglesAllZero.cpp. In the code there is a loop, which iterates the test 1e6 times and runs the assert command each iteration. Simply change 1e6 to 1 and you will run the simulation once.
