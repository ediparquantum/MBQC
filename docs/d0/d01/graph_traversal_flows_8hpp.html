Computes the future vertex in a 2D lattice graph based on a flow function.

For a 2D lattice graph, the flow function for a vertex returns the future vertex. The future vertex is calculated as presentVertex + numRows, where numRows is the number of rows in the lattice.

Parameters
presentVertexThe present vertex for which the future vertex is computed.
