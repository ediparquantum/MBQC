Applies rotation gates to a quantum register based on the provided qubit angles.

This function applies rotation gates to each qubit in the given quantum register qureg, using the corresponding angle from the qubitAngles vector. The rotation gate applied is the Z-rotation gate with the angle specified by qubitAngles[i] applied to the i-th qubit.

Parameters
quregThe quantum register to apply the rotation gates to.
qubitAnglesA vector of qreal values representing the rotation angles for each qubit.
Precondition
The size of qubitAngles must be equal to or smaller than the number of qubits represented by qureg.

Initializes a vector of qreal values with random angles that sum to a multiple of 2M_PI.

This function creates a vector of qreal values with a size of numQubits, where each element is initialized to a random fraction of X*M_PI/piDenominator, where X is uniformly distributed in the range [0, 2*piDenominator). The sum of the random fractions is adjusted to be a multiple of 2M_PI.

Parameters
numQubitsThe number of qubits.
piDenominatorThe denominator value used to generate the random angles.
Returns
A vector of qreal values with randomly generated angles that sum to a multiple of 2M_PI.
Precondition
The sum of the random fractions must be a multiple of 2M_PI.
Postcondition
The last element of the vector is adjusted to satisfy the sum requirement.
